## 附录

### A. 启动模拟器命令
```bash
# 方式1: 使用脚本
create-and-start-emulator.bat

# 方式2: 手动启动
C:\Android\Sdk\emulator\emulator.exe -avd Pixel_6_API_34

# 方式3: 后台启动
start /MIN C:\Android\Sdk\emulator\emulator.exe -avd Pixel_6_API_34
```

### B. 安装APK命令
```bash
# 安装APK
C:\Android\Sdk\platform-tools\adb.exe install installer\android\SecChat-debug.apk

# 查看已安装应用
C:\Android\Sdk\platform-tools\adb.exe shell pm list packages | findstr sec_chat

# 卸载应用
C:\Android\Sdk\platform-tools\adb.exe uninstall com.example.sec_chat
```

### C. 查看日志命令
```bash
# 查看所有日志
C:\Android\Sdk\platform-tools\adb.exe logcat

# 仅查看Flutter日志
C:\Android\Sdk\platform-tools\adb.exe logcat -s flutter

# 清空日志缓冲区
C:\Android\Sdk\platform-tools\adb.exe logcat -c

# 保存日志到文件
C:\Android\Sdk\platform-tools\adb.exe logcat > test_log.txt
```
